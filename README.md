# 📄 Automated Resume Screening Tool

## 📌 Overview
This project is an Automated Resume Screening Tool built using Python and NLP techniques. The system analyzes resumes, compares them with a job description, calculates matching scores, ranks candidates, and generates shortlist reports automatically.

The project simulates how Applicant Tracking Systems (ATS) work in real recruitment processes.

---

## 🚀 Features
- Resume PDF text extraction  
- Job description matching  
- TF-IDF vectorization  
- Cosine similarity scoring  
- Candidate ranking  
- Shortlist/reject decision  
- CSV report generation  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- PyPDF2  
- NLP Techniques
